# GOST-V3

A Bash script to install and manage GOST v3 tunneling services.

## Features
- Installs GOST v3 and dependencies.
- Manages services (add, remove, edit, start/stop/restart).
- Supports Foreign and Domestic server modes.

### Download and Run
